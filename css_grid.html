CSS Grid

What is Grid layout?

CSS Grid is magic and works just like you would think a layout does. It allows you to visually create a layout using the grid-template-areas, grid-template-rows, grid-template-columns rules. It is not restricted to just static layouts though. The grid-auto-rows, grid-auto-columns, and grid-auto-flow rules allow you to define how the browser should handle items beyond what is defined using a template rule. For example if you click a new item will be appended to the grid. Since this grid is defined with the following rules: 

            
grid-template-areas:
      "primary-nav header header header"
      "primary-nav main-content main-content main-content"
      "primary-nav footer footer footer";
    grid-template-columns: min-content min-content min-content fit-content;
    grid-auto-flow: row;
    grid-auto-rows: min-content;
    grid-auto-columns: 1fr;
The new items are placed along the bottom in equally sized and spaced boxes.
